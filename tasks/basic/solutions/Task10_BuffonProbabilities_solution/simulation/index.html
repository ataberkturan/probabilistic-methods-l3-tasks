Task 10 - Buffon's Needle Probabilities

Understand the geometric model first, then compare exact formulas with Monte Carlo estimates.

Model and variables

L = needle length d = distance between lines X = center distance to nearest line theta = angle of the needle
Sample-space idea One throw is one point (X, theta) inside the rectangle Ω = [0, d/2] × [0, pi/2].
Hit rule The needle hits a line when X <= (L/2) sin(theta).
